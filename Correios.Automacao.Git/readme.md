Este projeto de automação de testes funcionais consiste em validar o site dos correios com relação as funcionalidades de buscar CEP e endereços, utilizando as tecnoligias:
- Cucumber;
- Junit;
- FrameWork PageObjects;
- Selenium;
- WebDriverManager.
